En línea de comando podemos escribir el comando cat seguido del archivo que queremos inspeccionar, para ver lo que contiene este dentro.

Algunos comandos tiene un manual, ara abrir hay que escribir la palabra man (manual) antes del comando que queremos aprender a usar: man cat

El comando sudo indica que se quiere tener acceso de administrador a algún archivo.

Los sistemas operativos basados en Linux, como Ubuntu o Demian, tienen repositorios de dependencias para gestionar paquetes instalables. Ubuntu u otros softwares basados en Linux pueden acceder a estos repositorios para instalar las actualizaciones de las dependencias. Para ello se ocupa el siguiente comando: sudo apt-get update y para instalarlos utilizamos: sudo apt-get update

El comando para instalar node.js es: sudo apt install nodejs Sirve para escribir javascript en la terminal

El comando para instalar npm es: sudo apt install npm

El comando para instalar react.js es: npx create-react-app test
